这次写作大课总共分为五个部分：

1 / 为什么要写公众号文章

课程目标：写作与公众号，写作突破从0到1。

  • 认知：为什么写作？为什么写公众号？
  • 方法：讲故事，逻辑，传播性。
  • 运营：商业性，成长性，格局。

公众号写作的价值：现代职场底层的竞争力
公众号写作最厉害的功能，是积累所带来的复利效应。
沉淀内容，沉淀用户，形成超级个体闭环。

微信公众号让你与世界链接零距离。从表达到曝光，从曝光到流量，从流量到认知，从认知到用户，从用户到沉淀。

你写的不是文字，而是通往超级个体的最短路径！

2 / 如何写出一个好故事

好故事的共同特征是什么？GQ报道遵循的六条标准!

  • 一 是问题足够重要，要建立与时代的关系。
  • 二 是要展示复杂性，而不是孤立的一人一事。
  • 三 是选题要有想象力，不要模式化。
  • 四 是结论最好意外，有颠覆性。
  • 五 是要有好的故事情节。

  • 六 要有正确的价值观。

  • 好的故事具备超出讲清事情的野心，一定要建立共情效应。
    好故事应该聚焦于遭遇困境的人，麻烦缠身的人。
    好故事要揭露人物性格的本质，而人物性格的本质恰恰就是人物遭遇困境危机是得以凸显的。

    模仿是形成个人风格最有效的门径。反复修改润色故事，使其发生质变！

3 / 如何写出爆款文

风口深处是匠心，好文章不是写出来的。
产品思维，用户思维，要用做产品的诚意来写文章。
关键能力：有料，有诚意，有趣。充分的准备，热点的敏感度，超强的执行力
一切现象皆洞察。把握深度，人群，时效。但凡要火，必占一头。痛点：事业，生活，关系，认知；提高对生活和热点的敏感度。开头三件套：代入感，画面感，迫切感。好的标题是成功的基础，然后才是其他。

标题保证打开率，内容保证转发率！

4 / 从公众号写作到职场竞争力养成

找素材，专业文献，实践反思，行业观察。文章专业，首先是你要专业。创作思路：解决问题，深入浅出，观点鲜明。价值性，易读性，传播性。

5 / 如何运营好一个公众号

每个文字中都带有一种诚意在里面！

好公众号名字价值千金；多维度实现商业价值：媒体，电商，产品。文字有温度，有态度。有好奇，有利益。文章要全平台分发，尽可能多的引流。

这次课程价值几何，需要细细品味，人人皆不同！

为什么我喜欢孤独！

  • 曾经以为悲观抑郁都是因为孤独，后来才发现，孤独与我而言是食物和空气。

  • 我曾经是人群中的段子手，抖机灵耍滑稽随手就来，后来我变了，变得喜欢孤独！

1.怕麻烦

  • 什么人情世故，什么勾心斗角，什么该做不该做，什么对的错的，好的坏的，我都不在乎好么！烦不烦，统统滚蛋！你不觉得跟白痴无知对话是在浪费生命吗？好累，我要去睡一觉。

  • 酒桌什么的最煎熬了，KTV里DJ舞曲太可怕！享受一个人的晚风不比这些惬意得多么？半年前裸辞离开从大学到工作一直抱团的小团体，就是觉得一堆人太麻烦，所以一个人出走了。去陌生的城市，寻找我的孤独去了。

2. 因为懒

  • 懒得与别人交流。
  • 懒得去迎合别人的胃口。
  • 懒得逼自己融入一些群体。
  • 懒得忍受周围烦恼的声音。
  • 懒得去猜，去想念，去妒忌，去虚伪。
  • 懒得去装逼给别人看。

3. 一个人太特么自在了！

  • 一个人一口气看完几季的美剧而无人打扰，一个人在家营造氛围看个有内涵的电影哭得像个傻逼，别提有多爽，内心情感肆意挥洒！人多的时候能有这么真挚的情感流露吗？只有自己面对自己时，我才能彻底的放松，完全自由的面对自己和世界，而不是被各种人和事不断的分割。

  • 孤独不是孤单，我只是想把更多的时间用在感兴趣的事和人身上罢了。如果自己就是个丰富多彩且有趣的人，自己和自己玩就好了，何必浪费时间和无聊浅薄的人玩呢？（哼!我内心就是如此高傲）

  • 独处是人生中的美好时刻和美好体验。独处是灵魂生长的必要空间，在独处时，我们从别人和事务中抽身出来，回到了自己。

  • 一切严格意义上的灵魂生活都是在独处时展开的。和别人一起谈古说今，引经据典，那是闲聊和讨论；唯有自己沉浸于古往今来大师们的杰作之时，才会有真正的心灵感悟。和别人一起游山玩水，那只是旅游；唯有自己独自面对苍茫的群山和大海之时，才会真正感受到与大自然的沟通。—— 引自 周国平《独处也是一种能力》

GsonFormat

GitHub：GsonFormat

GsonFormat 说是开发中最常用的Android Studio插件绝不为过！

绝大多数的服务端都以json数据格式返回，而客户端需要根据api接口生成相应的实体类，这个插件

把这个过程自动化了，省了我们很多功夫不是！

GsonFormat生成实体类 的 快捷键 是： Alt + S 或者 Alt + Insert 也可以。

  • 1.首先在浏览器打开一条接口，然后使用 json.cn 把接口数据显示出来看看
  • 2.Ctrl + A 全选接口数据，Ctrl + C 复制
  • 3.回到Android Studio，Alt + S 快捷键弹出一个框框，然后 Ctrl + V 把刚刚复制的内容粘贴进去，点击OK即可生成你想要的数据接口的实体，是不是很酷啊！

android-butterknife-zelezny

GitHub-ButterKnife

  • android-butterknife-zelezny是根据butterknife定制的一款插件，能够方便快速初始化布局控件，再也不用手动 findViewById 了。
安装 :

  • 如何安装Android Studio的插件：
    以下以安装 android-butterknife-zelezny 为例子，其他的插件也是一样的安装步骤。

  • File Setting —> Plugins —> Browse repositories ,然后在输入框输入要安装的插件名称，点击Install即可安装，
    安装完之后按照提示重启AS便可使用。
